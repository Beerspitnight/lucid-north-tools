Onboarding Tools

Master Schedule Parser

Convert any Excel master schedule into a clean CSV for Harry Llama import. Handles multi-sheet workbooks, A/B day rotation, and auto-detects non-teaching periods like Prep, Planning, and Duty.

What it produces: CSV with Department, Teacher, and period columns — formatted for onboarding Step 3.

Open Parser

Quick steps:

  1. Upload .xlsx file
  2. Select the sheet to parse
  3. Configure non-teaching period labels
  4. Preview with validation warnings
  5. Download clean CSV
Validation features: Built-in checks for missing teacher names, empty schedules, duplicate teachers, and typo detection.

Bell Schedule & Calendar Parser

Extract bell schedules and school calendars from PDF or image files using AI. Outputs JSON formatted for Harry Llama onboarding Steps 2 and 7.

What it produces: Bell schedule JSON (period names, start/end times by schedule type) + Calendar JSON (holidays, PD days, early release dates, academic year boundaries).

Contact for Access
Note: Currently runs as a local desktop application. Requires Python 3 and a Z.ai API key.

Supported formats: PDF documents, scanned images. Automatically expands date ranges and maps categories to HLS classifications.

Expected Data Formats

Master Schedule CSV

Department, Teacher, Period 1 A Day, Period 1 B Day, Period 2 A Day, ... Math, Smith John, Algebra 1 (Room: 101), Geometry (Room: 101), ...

Bell Schedule JSON

{ "Regular Day": [ { "period_name": "Period 1", "start_time": "08:00", "end_time": "08:48" }, { "period_name": "Period 2", "start_time": "08:52", "end_time": "09:40" } ] }

Calendar JSON

{ "academic_year": { "student_start": "2025-09-04", "student_end": "2025-06-26", "teacher_start": "2025-08-30", "teacher_end": "2025-07-01" }, "events": [ { "date": "2025-09-01", "classification": "Holiday", "event_name": "Labor Day" }, { "date": "2025-11-13", "classification": "Early-Release", "event_name": "Staff PD" } ] }

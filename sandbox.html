Demo Flow Guide

Phase 1: Onboarding
  1. Select rotation type (A/B Day, 4-Day, etc.)
  2. Upload bell schedule — use Bell Schedule Parser from Tools page
  3. Upload master schedule — use Master Schedule Parser from Tools page
  4. Review teacher schedules and verify period assignments
  5. Configure period types and coverage preferences
  6. Add substitute pool
  7. Set calendar rules and holidays
Phase 2: Daily Operations
  1. Create teacher absences for today
  2. Run "Generate Coverage Plan"
  3. Review the coverage grid — note tiered assignments and fill rate
  4. Manually assign any remaining unfilled slots
  5. Confirm the plan
Phase 3: Reset

When demo data gets stale, request a sandbox reset to start fresh. Contact the dev team.
